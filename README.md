# 📚 Beyond

Beyond is a full-stack book e-commerce application built with ASP.NET Core, React and PostgreSQL.

This project is part of my software engineering portfolio and aims to simulate a real-world e-commerce platform while applying modern development practices.

#Tech Stack

#Backend
- ASP.NET Core
- C#
- Entity Framework Core
- PostgreSQL

#Frontend
- React
- Vite
- Axios

#Current Features

- Project setup
- REST API
- Swagger documentation

#Planned Features

- User authentication
- Book catalog
- Categories
- Shopping cart
- Checkout
- Orders
- Admin dashboard
