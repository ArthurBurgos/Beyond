# 📚 Beyond

> A modern full-stack book e-commerce built with ASP.NET Core, React and PostgreSQL.

## 📖 About

Beyond is a modern online bookstore designed to provide a complete e-commerce experience.

The project is being developed as a portfolio application with a focus on software architecture, clean code, REST APIs and modern full-stack development using ASP.NET Core and React.

---

## 🚀 Features

### Customer

- Browse books
- Search books
- Filter by category
- Shopping cart
- Checkout
- Order history
- Favorites
- User authentication

### Admin

- Manage books
- Manage categories
- Manage users
- Manage orders
- Dashboard

---

## 🛠 Tech Stack

### Backend

- ASP.NET Core 10
- C#
- Entity Framework Core
- PostgreSQL
- JWT Authentication
- Swagger

### Frontend

- React
- Vite
- TypeScript *(optional)*
- Axios
- React Router
